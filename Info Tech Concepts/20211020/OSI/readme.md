### L1 [實體層](https://zh.wikipedia.org/wiki/%E7%89%A9%E7%90%86%E5%B1%82) [Physical Layer](https://en.wikipedia.org/wiki/Physical_layer)
####規定
  - 為傳輸資料，需要"物理"鏈路與裝置的建立、維持、拆除，並具有機械的、電子的、功能的、規範的特性
####功能
  - 為資料端裝置提供傳送資料通路
  - 傳輸資料
